getzVar(self, var_in, retval=False, depend=False, keepEpoch=False, verbose=False)

source code 

get the contents of a zVar (or list of zVars) from the CDF file, they are saved in cdf.data

Parameters:
  • var_in (str) - name of varialbe to read (or list/tuple of names)
  • retval (bool) - return the data dont just add it to the dict
  • depend (bool) - also get the dependent zVars
  • keepEpoch (bool) - keep the cdf epoch, dont change it to Ticktock
  • verbose (bool) - be verbose to the screen
Returns: bool
True for success

Organization: Los Alamos National Lab

Contact: balarsen@lanl.gov

Version: V1: 20-Jul-2010 (BAL)

>>> cdf.getzVar('temp_e', depend=True)
