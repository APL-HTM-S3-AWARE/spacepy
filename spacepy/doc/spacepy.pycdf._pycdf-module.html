Function Details [hide private]

_compress(obj, comptype=None, param=None)

 

Set or check the compression of a CDF or Var

Parameters:
  • obj (CDF or Var) - object on which to set or check compression
  • comptype (ctypes.c_long) - type of compression to change to, see CDF C reference manual section 4.10. Constants for this parameter are in const. If not specified, will not change compression.
  • param (ctypes.c_long) - Compression parameter, see CDF CRM 4.10 and const. If not specified, will choose reasonable default (5 for gzip; other types have only one possible parameter.)
Returns: tuple
(comptype, param) currently in effect
