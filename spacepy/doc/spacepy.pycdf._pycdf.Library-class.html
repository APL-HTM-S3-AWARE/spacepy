Abstraction of the base CDF C library and its state.

Not normally intended for end-user use. An instance of this class is created at package load time as the lib variable, providing access to the underlying C library if necessary.

Calling the C library directly requires knowledge of the ctypes package.

__init__ searches for and loads the C library; details on the search are documented there.

Raise exception or warning based on return status of CDF call

Parameters:
  • status (int) - status returned by the C library, equivalent to CDFStatus
  • ignore (sequence of ctypes.c_long) - CDF statuses to ignore. If any of these is returned by CDF library, any related warnings or exceptions will not be raised. (Default none).
Returns: int
status (unchanged)
Raises:
  • CDFError - if status < CDF_WARN, indicating an error
  • CDFWarning - if CDF_WARN <= status < CDF_OK, indicating a warning, and interpreter is set to error on warnings.

call(self, *args, **kwargs)

source code 

Call the CDF internal interface

Passes all parameters directly through to the CDFlib routine of the CDF library's C internal interface. Checks the return value with check_status.

Parameters:
  • args (various, see ctypes.) - Passed directly to the CDF library interface. Useful constants are defined in the const module of this package.
  • ignore - sequence of CDF statuses to ignore. If any of these is returned by CDF library, any related warnings or exceptions will not be raised.
Returns: int
CDF status from the library
Raises:
  • CDFError - if CDF library reports an error
  • CDFWarning - if CDF library reports a warning and interpreter is set to error on warnings.

Note: Terminal NULL_ is automatically added to args.
