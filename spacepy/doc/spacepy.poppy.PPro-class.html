        PPro

PoPPy point process object

Initialize object with series1 and series2. These should be timeseries of events, given as lists, arrays, or lists of datetime objects. Includes method to perform association analysis of input series

Output can be nicely plotted with plot method

Author:

Steve Morley, Los Alamos National Lab, smorley@lanl.gov Jonathan Niehof, Los Alamos National Lab, jniehof@lanl.gov

source code 

Create basic plot of association analysis.

Uses object attributes created by the assoc method and, optionally, aa_ci.

Parameters:
  • figsize - passed through to matplotlib.pyplot.figure
  • dpi - passed through to matplotlib.pyplot.figure
  • asympt (bool) - True to overplot the line of asymptotic association number
  • show (bool) - Show the plot? (if false, will create without showing)
  • norm (bool) - Normalize plot to the asymptotic association number
