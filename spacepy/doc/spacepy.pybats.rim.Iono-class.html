A class for handling 2D output from the Ridley Ionosphere Model. Instantiate an object as follows:

>>>iono = rim.Iono('filename.idl')

...where filename.idl is the name of a RIM 2D output file.

add_cont(self, ax, var, n=24, maxz=False, lines=True, cmap=False, add_cbar=False, **kwargs)

source code 

Create a polar contour of variable 'var' and add the resulting artist to axis 'ax'. Make sure that 'ax' is a polar axes or you'll be sorry.

Available kwargs: n: Set number of levels. Should be a multiple of 3 for best match between filled and traced contours. Default is 21.

lines: Add unfilled black solid/dashed contours to plot for additional contrast. Default is true.

maxz: Set the max/min value for the color bar. Default is set by data.

cmap: Set the colormap. Default is to autoselect using classic IE maps.

add_cbar: Add colorbar to plot. Default is to not add one to the plot.

Extra keywords are passed to the contourf routine.
