 3  """pyCDF: Python interface to NASA's Common Data Format library 
 4   
 5  This package provides a Python interface to the Common Data Format (CDF) 
 6  library used for many NASA missions, available at U{http://cdf.gsfc.nasa.gov/}. 
 7  It is targeted at Python 2.6+ and should work without change on either 
 8  Python 2 or Python 3. 
 9   
10  The interface is intended to be quite 'pythonic' rather than reproducing the 
11  C interface. To open or close a CDF and access its variables, see the L{CDF} 
12  class documentation. Accessing data within the variables is via the L{Var} 
13  class. 
14   
15  The base CDF library must be properly installed in order to use this package. 
16  pycdf will search for the library in this order: 
17    1. A directory named by the environment variable CDF_LIB (which should be 
18       set if using the definitions file provided with the CDF library). 
19    2. A subdirectory C{lib} in a directory named by the environment variable 
20       CDF_BASE. 
21    3. The standard system library search path. 
22  If pycdf has trouble finding the library, try setting CDF_LIB before importing 
23  the module, e.g. if the library is in CDF/lib in the user's home directory:: 
24    import os 
25    os.putenv("CDF_LIB", "~/CDF/lib") 
26    import pycdf 
27   
28  The L{const} module contains constants useful for accessing the underlying library. 
29  """

32  __author__ = "Jonathan Niehof <jniehof@lanl.gov>"
