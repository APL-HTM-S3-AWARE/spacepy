13      convert dtype data into GPS[s] number 
14   
15      Possible data types: 
16      ISO: ISO standard format like '2002-02-25T12:20:30' 
17      UTC: datetime object with UTC time 
18      TAI: elapsed seconds since 1958/1/1 (includes leap seconds) 
19      UNX: elapsed seconds since 1970/1/1 (all days have 86400 secs of occasionally unequal lenghts) 
20      JD: Julian days elapsed 
21      MJD: Modified Julian days 
22      RTD: Rata Die days elapsed since 1/1/1 (MatPlotLib serial time) 
23      CDF: CDF epoch time: milliseconds since 1/0/0 00:00:00.000 
24      GPS: GPS epoch is 0000 UT (midnight) on January 6, 1980 (includes leap seconds) 
25   
26      Input: 
27      ====== 
28          - data (int, datetime, float, string as singles or array) : time stamp 
29          - dtype (string) : data type for data ISO, UTC, TAI, UNX, JD, MJD, RTD, CDF, GPS 
30   
31      Returns: 
32      ======== 
33          - GPS (int as single or array) : elapsed seconds since 1958/1/1 (includes leap seconds, i.e. all secs have equal lengths) 
34   
35      Example: 
36      ======== 
37   
38      >>> x=datetime.datetime(2002,2,25,12,20,30) 
39      >>> getGPS(x, 'UTC')
